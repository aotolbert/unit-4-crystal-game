The Crystal Collector Game!

Hey there!! Thanks for stopping by to play Crystal Collector with me. The game is pretty simple.

-I will choose a random number that will become the "Target Score" as well as assigning hidden values to the crystals you see to the right.

-The object of the game is to discern what values each crystal holds and add them together in a way that allows your score to match the target score exactly but be careful, if you go over, even by just one, you lose!!

-Click on any crystal to begin!
